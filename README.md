## Project Description

A work-in-progress inventory management system designed for a café environment. The system supports role-based access for four user levels: **Barista**, **Shift Supervisor**, **Store Manager**, and **Superadmin**, each with different permissions and responsibilities.

## Tech Stack

- **Programming Language:** VB.Net  
- **Database:** Microsoft Access
